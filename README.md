# PicPay Simplificado

API REST desenvolvida em **Java + Spring Boot** como solução para o desafio técnico de back-end do [PicPay](https://www.picpay.com?utm_source=chatgpt.com), simulando uma plataforma de pagamentos digitais com transferências financeiras entre usuários.

## Tecnologias Utilizadas

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* Banco de Dados Relacional
* APIs REST
* Maven

## Funcionalidades

* Cadastro e autenticação de usuários
* Autorização segura com JWT
* Transferência de saldo entre contas
* Validação de regras de negócio para transações
* Persistência de dados em banco relacional
* Tratamento de exceções e respostas padronizadas

## Arquitetura e Conceitos Aplicados

O projeto foi estruturado seguindo boas práticas de desenvolvimento back-end, incluindo separação em camadas (**Controller, Service, Repository e Entity**), visando maior escalabilidade, manutenção e organização do código.

Durante o desenvolvimento, foram aplicados conceitos importantes como:

* Segurança de APIs
* Programação orientada a objetos
* Validação de dados
* Modelagem relacional
* Boas práticas RESTful

## Objetivo

Este projeto teve como objetivo aprofundar conhecimentos em desenvolvimento back-end com foco em sistemas transacionais, segurança e construção de APIs robustas para aplicações financeiras.
